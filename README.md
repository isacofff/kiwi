# Guide

-Binds

To bind a module, right click on the selected module while holding shift to enter the binding mode.
Press Backspace to exit, and any other charchter to save the bind.

-ClickGui

Press Right Shift to enter the Gui, navigate it by clicking the above tabs to change the window.
Click the SearchBar and start typing to search through the Gui.
